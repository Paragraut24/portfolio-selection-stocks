Complexity Analysis

Algorithm Time Complexity Space Complexity Optimal Solution
2D DP O(n * W) O(n * W) Yes
1D DP O(n * W) O(W) Yes
Greedy O(n log n) O(n) No (Heuristic)
Fractional O(n log n) O(n) Yes (for fractional)

Algorithm Recommendations

When to use 2D DP:

Need optimal solution with detailed tracking. Can afford O(n*W) space.

When to use 1D DP:

Need optimal solution but memory is limited. Best space-time tradeoff.

When to use Greedy:

Need fast approximate solution. Real-time applications with large datasets.

When to use Fractional:

Partial stock purchases allowed. Provides upper bound for 0/1 knapsack.
