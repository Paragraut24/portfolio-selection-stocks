Algorithm Information

2D Dynamic Programming (0/1 Knapsack)

Optimal solution using 2D DP table. Each stock selected at most once. Time: O(n*W), Space: O(n*W)

1D Dynamic Programming (0/1 Knapsack)

Space-optimized DP solution. Each stock selected at most once. Time: O(n*W), Space: O(W)

Unbounded Knapsack

Allows multiple purchases of same stock for concentrated portfolios. Time: O(n*W), Space: O(W)

Greedy Algorithm

Fast heuristic based on return/cost ratio. Time: O(n log n), Space: O(n)

Fractional Knapsack

Allows partial stock purchases (theoretical). Time: O(n log n), Space: O(n)
